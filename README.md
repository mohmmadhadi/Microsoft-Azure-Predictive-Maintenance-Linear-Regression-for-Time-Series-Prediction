# Microsoft-Azure-Predictive-Maintenance-Linear-Regression-for-Time-Series-Prediction

Today, it is impossible to store and transfer data without using cloud platforms. Microsoft
Azure is one of the companies providing public cloud computing services. In the past, it was
known as Windows Azure, which was later changed to Microsoft Azure due to its greater
efficiency than Windows. Windows Server Azure is a cloud platform managed by Microsoft
that is used to develop application management over the global network.
According to the explanations provided, in this project we plan to use Microsoft Azure data
from our machines and their downtimes to launch predictive net using linear regression and
using PySpark library in Python.
Keywords: Microsoft Azure, cloud computing, maintenance, linear regression, PySpark.
